# 首页布局与响应式优化

## 背景
- 首页 `Complete Tissue Product Portfolio` 三卡区块误挂了通用等高类，但缺少列数修饰，导致横向三列布局被破坏。
- 首页 `Who We Are` 区块仍使用绝对定位背景图与右侧偏移卡片的组合，容易在不同分辨率下出现左右不对齐。
- 当前主题已形成稳定基调，但仍需要补强首页及关键断点下的响应式适配。

## 目标
- 恢复首页三张产品能力卡在桌面端的横向三列布局，同时保持平板和手机端自适应。
- 将首页 `Who We Are` 区块调整为更稳定的双栏图文布局，保证不同分辨率下对齐自然。
- 顺手检查并补强关键页面在常见断点下的间距、对齐和可读性。
