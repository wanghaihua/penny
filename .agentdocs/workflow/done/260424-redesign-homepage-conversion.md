# 首页商业转化改版

## 背景
- 用户希望基于现有网站内容，重新设计更美观、更有商业转化力的首页。
- 风格要求为现代、简洁、高级、可信，目标是提升用户咨询/购买转化。

## 设计方向
- 首屏从模板式大图文改为“采购落地页”结构：明确价值主张、产品视觉、资历证据与报价入口同时出现。
- 首页内容顺序围绕买家决策组织：核心承诺 -> 可信证据 -> 产品选择 -> 订单流程 -> 询盘 CTA。
- 视觉保持项目既有明亮、绿色、纸张感、纯色克制风格，不引入新框架或后端能力。
