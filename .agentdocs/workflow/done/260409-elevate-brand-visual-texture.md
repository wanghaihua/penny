# 品牌质感增强

## 背景
- 当前站点的结构与主题已经稳定，但整体观感仍偏平，绿色理念表达主要停留在配色层，缺少更具体的品牌符号、标签层次和动态反馈。
- 用户希望在不破坏商务、专业、可持续主基调的前提下，增加绿色相关元素、标签、logo 细节与轻量动画，让网站更有质感和设计感。

## 目标
- 保持明亮、纯色、无渐变的前提下，为正式页面补充更鲜明的绿色品牌细节。
- 增强共享 logo、首页首屏与关键内容区的层次感，让页面不再显得单调。
- 添加克制、轻量、跨设备稳定的入场动画和悬浮反馈，不引入花哨或性能负担明显的效果。
