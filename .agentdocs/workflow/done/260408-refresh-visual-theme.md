# 视觉主题升级

## 背景
- 站点已具备较完整的业务内容，但视觉语言仍偏模板改造，品牌识别不够集中。
- 当前页头、卡片、CTA、产品规格和页脚虽已统一结构，但颜色层次、材质感和商业可信度仍有提升空间。
- 需要在不增加前端复杂度的前提下，形成更专业、可持续、商业化的整体风格。

## 目标
- 基于现有结构建立统一的品牌主题色与界面层次。
- 让首页、服务页、产品页、联系页与页脚在视觉上更连贯、更成熟。
- 重新检查三语翻译数据，确认结构完整、无空值、无缺键。

## 实施阶段
1. 盘点现有样式与核心视觉区块
2. 统一主题变量与核心组件样式
3. 补充联系页、产品页、页脚等页面气质
4. 执行翻译与本地校验
