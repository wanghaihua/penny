# 内容表达现代化改造

## 背景
- 用户希望基于现有静态多语言官网进行修改，使内容表达更现代化、更有创意。
- 本轮优先落地前次分析中对品牌感与询盘转化影响最大的局部改造，不引入新框架或后端能力。

## 改造范围
- 首页：将首屏表达从产品罗列升级为采购确定性表达。
- 关于页：校准可信度数字，并将通用使命愿景改为更贴近纸品外贸采购的能力表达。
- 服务页：把三步流程升级为更完整的订单旅程。
- 产品页：让产品卡片直接呈现采购关键信息与询盘入口。
- 联系页：把自由留言表单增强为结构化询盘生成器，继续使用 `mailto:`。
- 多语言：同步 `translations/*.json` 与 `js/embedded-translations.js`。
