# 主题与翻译细化

## 背景
- 当前站点已经完成一轮主题统一，但样式中仍大量使用渐变，整体氛围偏重，不完全符合“专业、可持续、商业化且不过暗”的目标。
- 首页与服务页的多组信息卡片高度不一致，影响内容区块的秩序感与商业呈现。
- `about.html` 的客户评价区仍沿用模板式轮播结构，卡片与背景关系松散，间距也偏紧。
- 中文翻译新增字段出现乱码问号，且 `js/embedded-translations.js` 需要与 JSON 真源保持同步。
- 语言切换器已接入共享布局，但视觉与响应式表现仍可进一步收敛和简化。

## 目标
- 将全站主题收敛为无渐变、明亮、专业、强调可持续气质的纯色体系。
- 统一首页与服务页目标卡片的高度与留白，提升版面协调性。
- 重设计关于页客户评价区，使背景、卡片和间距更统一。
- 修复中文翻译异常，并确保三语 JSON 与 embedded fallback 完整同步。
- 重设计语言切换器，使其更简洁、优雅，并兼容桌面与移动端。

## 实施阶段
1. 记录并对齐本轮设计与翻译约束
2. 重构主题样式与共享交互视觉
3. 调整页面结构与卡片布局
4. 修复翻译并执行本地校验
