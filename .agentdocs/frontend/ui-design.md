# 界面设计约束

## 品牌气质
- 站点整体视觉应服务于纸品外贸业务表达，关键词为：专业、可持续、商业化、可信赖。
- 页面整体保持明亮、轻稳，不使用大面积深色压迫感，也不使用花哨渐变营造“模板感”。

## 配色原则
- 主色使用自然感的森林绿或鼠尾草绿，传达稳定供应与可持续属性。
- 大面积背景使用纸张感米白、暖白或浅灰绿，保持洁净和产品行业相关性。
- 强调色仅作为小面积点缀或 CTA 使用，优先采用克制的铜金或暖棕，不喧宾夺主。
- 除图片本身外，界面层尽量使用纯色，不新增渐变背景、渐变按钮或渐变分隔。

## 组件原则
- 首页、服务页、关于页中同层级卡片应保持统一高度、圆角、边框和阴影强度。
- 多卡片编排优先使用统一网格而不是依赖单卡 `margin-bottom`，桌面端保持稳定列数，平板端自动降为两列，手机端降为一列。
- 首页或落地页中的图文介绍区优先使用真实双栏结构，不依赖背景图与偏移卡片制造对位关系，以减少不同分辨率下的错位风险。
- 卡片之间优先通过稳定留白和边框层次建立关系，不依赖过重投影或复杂背景。
- 语言切换器应保持轻量、语义清晰、移动端可点击区域充足；当前优先采用常驻分段按钮，不使用下拉菜单。
- 共享头部顶栏中的社交、电话、邮箱与语言切换应视为一组统一控件：社交入口优先用轻量图标，电话/邮箱优先用无背景的图标加文字，语言切换器保留为更窄的分段胶囊，并允许在平板和手机端自然换行居中。
- 页脚、CTA 和统计区可以使用比正文更深的纯色底，但需避免接近纯黑的视觉重量。
- 评论类内容优先使用单卡轮播或单列展示，避免在窄空间里并排堆两张卡片导致内容拥挤。
